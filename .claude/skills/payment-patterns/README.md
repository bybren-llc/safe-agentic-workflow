# Payment Patterns

Polar payment integration patterns for ConTStack subscriptions and checkouts.

## Purpose

Routes to existing payment patterns and provides evidence templates for payment testing.

## Key Topics

- Webhook signature verification
- Checkout session creation
- Idempotency handling
- Multi-tenant payment isolation
- Test mode enforcement

## Usage

This skill is automatically invoked when working on:
- Payment integrations
- Subscription management
- Webhook handlers
- Checkout flows
