![Status](https://img.shields.io/badge/status-production-green)

> Agentic orchestration patterns for long-running tasks. Implements evidence-based delivery and Simon Willison's agent loop.

## Quick Start

This skill activates automatically when you:
- Orchestrate multi-step implementation tasks
- Manage work across multiple subagents
- Run long-running sessions that need checkpoints
- Prepare PRs for merge (mandatory QAS gate)

## What This Skill Does

Codifies evidence-based delivery and iterative agent loops for orchestrating complex, long-running tasks. Ensures verifiable progress through checkpoints and intelligent escalation when blocked.
